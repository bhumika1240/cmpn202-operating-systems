# Coursework Image Evidence

This directory contains screenshots used as evidence for the CMPN202 Operating Systems coursework.

Each subfolder corresponds to a specific week and contains:
- Screenshots collected during lab activities
- A README file explaining the purpose of each image

## Folder Structure

- `week1/` – System inspection and basic Linux commands
- `week2/` – Process management and system monitoring
- `week3/` – User management and permissions
- `week4/` – Network configuration and firewall setup
- `week5/` – Security hardening and server monitoring

All screenshots directly support the explanations provided in the respective `weekX.md` files.
