# Image Evidence Directory

This directory contains screenshots used as evidence for the CMPN202 Operating
Systems coursework.

Screenshots are organised by week to make it easy to locate evidence for each
phase of the coursework.

## Folder Structure

- `week1/` – System planning, OS inspection, and basic Linux commands
- `week2/` – Process management and system monitoring
- `week3/` – Users, permissions, and networking
- `week4/` – SSH hardening, firewall configuration, and updates
- `week5/` – Advanced security controls and monitoring

Each subfolder includes a README file describing the screenshots it contains.
